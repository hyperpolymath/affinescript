= AffineScript Testing Report (retired)
:icons: font

[IMPORTANT]
====
This file has been *retired*.  It was a one-off snapshot dated
2025-12-29 reporting "47 PASS / 27 FAIL / 63.5% pass rate", a
description that has been contradicted by the live CI gate since
early 2026 (the actual gate has been at >250 tests / 0 failures for
the entire first half of 2026).

Continuing to ship a stale snapshot as if it were authoritative is a
DOC-09 over-claim hazard ("reject stale-as-current status"), so it
is retired in place rather than left to mislead.
====

== Why kept at all

Removing the file would break inbound links from older docs and
external citations.  Replacing the body with a redirect preserves
the URL while making the retirement loud.

the same snapshot) is retired on the same grounds and points here.
