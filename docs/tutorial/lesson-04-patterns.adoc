= Lesson 4: Pattern Matching

Destructure data with match expressions:

[source,affinescript]
----
fn describe(x: Int) -> String {
  return match x {
    0 => "zero",
    1 => "one",
    _ => "many",
  };
}
----

== Tuple Patterns

[source,affinescript]
----
fn first(pair: (Int, Int)) -> Int {
  return match pair {
    (a, b) => a,
  };
}
----
