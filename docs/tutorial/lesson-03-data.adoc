= Lesson 3: Data Structures

== Tuples

Group multiple values:

[source,affinescript]
----
fn make_pair() -> (Int, Int) {
  return (1, 2);
}
----

== Records

Named fields:

[source,affinescript]
----
fn make_point() -> Point {
  return { x: 10, y: 20 };
}
----

== Arrays

Collections of same type:

[source,affinescript]
----
fn make_list() -> [Int] {
  return [1, 2, 3];
}
----
