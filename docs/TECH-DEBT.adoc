== Update protocol

* One PR per ledger unit; `Refs #N` (never `Closes` on the #175 epic — only
  the owner closes it).
* Every capability-affecting PR updates `CAPABILITY-MATRIX.adoc` *and* this
  ledger in the same change; `STATE.a2ml` mirrors after.
* `dune runtest --force` must stay green (baseline 260, 2026-05-19).
* typed-wasm ABI changes are multi-producer: coordinate with
  `hyperpolymath/typed-wasm` and `hyperpolymath/ephapax`.
