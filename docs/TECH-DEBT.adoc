====

ecosystem epic (issue #175). Sub-issues are the work units; this ledger is the
critical path and the shared state across sessions/bots. ID classes:
`DOC` (documentation truthing), `CORE` (compiler soundness/completeness),

#228 / #177 but never committed). For the spine + contract see

link:CAPABILITY-MATRIX.adoc[CAPABILITY-MATRIX.adoc].
