as explicit parameters. `Console.log` (24 files, mostly vm tests) is trivially fixed by routing
to an I/O port. `Date.now` / `Math.random` must be fed as explicit inputs to the wasm brain.

== The 4-gate recipe (every kernel, every wave)

[[recipe]]

[arabic]
. *Re-decompose* — NOT transliterate: collapse incidental Promise/async, thread service-locator state as params, enum the options, hand-ladder int rendering, keep wire bytes identical.
. *`affinescript compile` → wasm* — gate 1: builds.
. *`affine-parity` oracle-vs-wasm sweep green* — gate 2: parity.
. *`echo-boundary` proves each host-boundary encoding LOSSLESS or declared-clamp* — gate 3: faithful.
. *`affine-assail` clean* — no undeclared clamp / unguarded decoder — gate 4.

Each migrated kernel is staged under `proposals/idaptik/migrated/<name>/` with `.affine` + harness + evidence.
