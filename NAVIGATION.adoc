== Common Tasks

=== Building the Compiler

[source,bash]
----
cd .build/
just build           # Build compiler
just test            # Run tests
just install         # Install locally
----

=== Running Tests

[source,bash]
----
cd .build/
just test            # Run all tests
just test-conformance # Run conformance tests
----

=== Development

[source,bash]
----
cd .build/
just dev             # Development mode
just watch           # Watch mode for auto-rebuild
just fmt             # Format code
----

== Document Conventions

* **`.adoc`** - AsciiDoc format (preferred for complex docs)
* **`.md`** - Markdown format (for GitHub compatibility)
* **`.scm`** - Guile Scheme (machine-readable metadata)
* **`.ml`** - OCaml source code
